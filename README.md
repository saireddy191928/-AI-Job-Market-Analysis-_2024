# 🤖 AI Job Market Analysis (2024–2030) – Exploratory Data Analysis

## 📌 Project Overview

This project presents an **Exploratory Data Analysis (EDA)** of an **AI Job Market dataset covering the years 2026–2030**. Using Python and modern data visualization libraries, the analysis explores salary trends, education requirements, experience levels, industries, locations, and automation risks within the AI job market.

The goal is to transform raw data into meaningful insights that help understand how the AI employment landscape is expected to evolve during the 2026–2030 period.

---

# 🎯 Objectives

* Analyze salary trends across AI job roles.
* Compare salaries by education level and experience.
* Identify the highest-paying AI jobs.
* Explore automation risk across professions.
* Understand industry and location-based salary variations.
* Generate business insights through visual analytics.
